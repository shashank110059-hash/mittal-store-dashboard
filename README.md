# mittal-store-dashboard
A modern AI-powered Review Dashboard built with Next.js 16, React 19, Tailwind CSS, Framer Motion, and Google Gemini AI. Designed for Mittal Store to showcase customer reviews, generate AI-powered review suggestions, and provide an elegant, responsive interface for managing customer feedback.
# 🛕 Mittal Store Review Dashboard

A modern and responsive **AI-powered Review Dashboard** built for **Mittal Store**, a spiritual and devotional products store specializing in:

- 🛕 Laddu Gopal Ji Poshak
- 🌺 Mata Rani Poshak
- 🪔 Puja Samagri
- 🛍️ Religious Idols (Murtis)
- 🪙 Brass Diyas
- 🙏 Temple Accessories

The dashboard helps showcase customer reviews while leveraging **Google Gemini AI** to generate natural review suggestions.

## 🚀 Getting Started

Clone the repository

```bash
git clone <repository-url>
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```
